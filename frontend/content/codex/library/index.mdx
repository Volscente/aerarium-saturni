# Library

Curation logic for external resources. The Library is a curated index of high-quality external materials - books, videos, courses, and newsletters - selected for their depth, accuracy, and relevance to long-term investing.

## Contents

- **Media** - YouTube channels, Blogs, and Newsletters.
- **Education** - Tracker for courses followed and certifications.
- **Reading List** - Bibliographies and links to source material.
