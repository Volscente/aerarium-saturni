# The Codex

**The Codex** is the centralized knowledge base for the Aerarium Saturni platform. It documents financial theory, Greeks, pricing models, and platform-specific logic with first-class support for mathematical notation.

## Features

- **LaTeX rendering** — inline formulas like $e^{i\pi} + 1 = 0$ and full display equations, rendered via KaTeX at build time.
- **Structured navigation** — articles organized by topic so analysts can move quickly between related concepts.
- **Full-text search** — header-prioritized indexing returns results for financial terms in under 200 ms.
